    一 连接MySQL

    1.安装pymysql

    在django中使用MySQL依赖于pymysql库

    在文件目录下 pip install pymysql

    2.配置数据库

    settings.py->设置

    在mysql-连接信息里查看

    3.启用pymysql

    _ init_.py

    二 创建数据库

    1.创建模块

    python manage.py startapp + 名字

    2.启用app

    settings.py

    可直接引用

    3.创建用户模型

    models.py

    django 全部model类型参考：https://www.cnblogs.com/wt869054461/p/4014271.html

    • class 表名(modles.Model):
    • AutoField 自增
    • primary_key=True 可空
    • CharField(max_length=) char类型
    • null=True不能为空

    4.确认，自动建表

    在文件根目录下执行命令

    有manage.py文件的就是根目录

    python manage.py makemigrations
    python manage.py migrate

    建好了

    有些表是django自动建的
